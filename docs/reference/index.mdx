# Reference guides

## What to expect

This section provides the reference information for conda-forge resources.
Their main goal is to compile all the raw data you may need to consult as part of your maintainer operations.
These guides don't try to be explanations or tutorials to help you understand a topic better.

For more information, read [The difference between reference and explanation](https://diataxis.fr/reference-explanation/).
