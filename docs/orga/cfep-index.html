Index of CFEPs¶

All major changes to the project should be submitted as conda-forge enhancement proposals (CFEP). These CFEPs are similar to Python’s PEP and IPython’s IPEP processes.

CFEPs are stored in the cfep GitHub repository. Below is a list of conda-forge enhancement proposals (CFEP):
